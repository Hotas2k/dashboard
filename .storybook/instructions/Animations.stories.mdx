# Animations

Using [Framer Motion](https://www.framer.com/motion/).

## Example

```tsx
import { motion } from 'framer-motion';

const Component = () => (
  <motion.div
    animate={{
      scale: [1, 2, 2, 1, 1],
      rotate: [0, 0, 270, 270, 0],
      borderRadius: ['20%', '20%', '50%', '50%', '20%'],
    }}
  />
);
```
