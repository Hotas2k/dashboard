# Form

We are using [React Hook Form](https://react-hook-form.com/) when creating forms. When validating we're using [validator.js](https://www.npmjs.com/package/validator) and some homemade regex.

## Example setup

See more examples [here](https://entire-storybook.internal.isdemo.se/?path=/story/documentation-form-components--page).

```tsx
import { TextField } from '@entire.se/components';
import { useForm } from 'react-hook-form';

import { useValidate } from 'hooks';

const Component = () => {
  const { isEmail } = useValidate();
  const {
    register,
    handleSubmit,
    formState: { errors },
    reset
  } = useForm();

  const onSubmit = handleSubmit((data) => {
    console.log(data);
    reset();
  });

  return (
    <form onSubmit={onSubmit}>
      <TextField
        name="email"
        type="email"
        label="Email"
        register={register}
        validation={{
          required: true,
          validate: isEmail
        }}
        error={errors.email}
      />
      <button type="submit">Submit</button>
    </form>
  );
};
```
